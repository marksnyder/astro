What Is Astro?

Astro is a self hosted layer for building applications on top of AI: agents, scheduled tasks, retrieval memory, tables, feeds, and a full MCP API so your models can read and write real data. Universes separate projects so each app or team gets its own boundary without mixing agents or content.

Core Features

Agents, retrieval, MCP tools, and data planes in one place so you can ship AI backed experiences.

Vector Store

All markdowns, documents, and action items are automatically embedded into a local ChromaDB vector store using CPU based embeddings. No API keys required. Semantic search across your entire knowledge base.

Markdowns

Structured notes with rich formatting, embedded images, categories, and full text search. Every markdown is included in AI retrieval context automatically.

Documents

Upload PDF, DOCX, XLSX, PPTX, TXT, MD, and CSV files. Documents are automatically ingested and embedded so they become queryable memory, not just storage.

Diagrams

Create and edit visual diagrams powered by Excalidraw. Draw shapes, arrows, and freehand sketches with a full embedded editor. Import and export .excalidraw files, and let AI agents create diagrams programmatically via MCP.

Tables

Build structured data with spreadsheet style tables. Define typed columns (text, number, boolean), manage rows with inline editing, and import or export CSV. Full MCP integration lets agents create and query tables programmatically.

Action Items

Track tasks with priority flags, due dates, and categories. Tasks integrate directly into your knowledge system and are included in AI retrieval.

Bookmarks

Save links with titles, URLs, and categories. Links join your searchable knowledge base for humans and agents.

Agent Network

A built in IRC server enables communication with AI agents across platforms. Coordinate agents, communicate across systems, and integrate with external AI platforms from one central hub.

Agent Tasks

Point tasks at a markdown note and an IRC channel: run on demand, on a cron schedule, or once at a future time. Instructions are sent through a dedicated runner with a configurable template (default includes API read URL and markdown ID). Manage tasks in the app or via MCP: list_agent_tasks, read_agent_task, write_agent_task, update_agent_task, delete_agent_task, and run_agent_task_now.

Feeds

Ingest data from external services through authenticated API endpoints. Push markdown or files into a timeline view. Ideal for CI reports, monitoring alerts, and automated agent output.

MCP Server

A built in Model Context Protocol server exposes 49 tools covering search, notes, tables, diagrams, tasks, categories, links, documents, feeds, and agent tasks. Any MCP-compatible AI agent can read, write, and search your knowledge base.
