What Is Astro?

Astro is an AI operating system for managing information, agents, prompts, tasks, and data feeds — alongside markdowns, documents, and links. Everything lives inside Universes — cleanly separated workspaces where structure keeps you organized while agents accelerate how you work.

Core Features

One system to run agents, prompts, and knowledge — with structure and retrieval built in.

Vector Store

All markdowns, documents, and action items are automatically embedded into a local ChromaDB vector store using CPU-based embeddings — no API keys required. Semantic search across your entire knowledge base.

Markdowns

Create and organize structured markdowns with rich formatting, embedded images, category assignment, and full-text search. Every markdown is included in AI retrieval context automatically.

Documents

Upload PDF, DOCX, XLSX, PPTX, TXT, MD, and CSV files. Documents are automatically ingested and embedded so they become queryable memory — not just storage.

Action Items

Track tasks with priority flags, due dates, and categories. Tasks integrate directly into your knowledge system and are included in AI retrieval.

Bookmarks

Save and organize links inside your Universe with titles, URLs, and categories. Links become part of your searchable knowledge base.

Agent Network

A built-in IRC server enables communication with AI agents across platforms. Coordinate agents, communicate across systems, and integrate with external AI platforms from one central hub.

Prompts

Organize reusable message templates in a visual board with custom categories and emoji labels. Drag-and-drop prompts between categories, schedule with cron, or run on demand.

Feeds

Ingest data from external services through authenticated API endpoints. Push markdown or files into a timeline view — ideal for CI reports, monitoring alerts, and automated agent output.

MCP Server

A built-in Model Context Protocol server exposes 23 tools — search, notes, tasks, categories, links, documents, and feeds — so any MCP-compatible AI agent can read, write, and search your knowledge base.

Diagrams

Create and edit visual diagrams powered by Excalidraw. Draw shapes, arrows, and freehand sketches with a full embedded editor. Import and export .excalidraw files, and let AI agents create diagrams programmatically via MCP.
