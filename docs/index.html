Everything you need. Nothing you don't.

One container. Zero cloud dependencies. Complete control.

AI Chat with RAG

Ask questions and get answers grounded in your own documents. Powered by OpenAI with support for GPT-5, o3, and more — switch models on the fly. Toggle between RAG-augmented and direct chat with one click.

Notes & Documents

Rich notes with image embedding, plus a full document archive supporting PDF, DOCX, XLSX, and PPTX. Every document is automatically chunked and indexed into the vector store for AI retrieval.

Action Items

Track tasks with priority flags, due dates, and category organization. Link action items to related notes or documents. Everything is searchable through the vector store.

Autonomous Agents

Define AI team members with custom profiles and instructions. Create multi-step activities that run on a schedule or on demand — delegate research, analysis, and reporting to your agents.

Links & Bookmarks

Save and organize bookmarks with categories. A companion browser extension lets you save links from any page with one click. Pin your most important items to the header bar for quick access.

Self-Hosted & Secure

Runs entirely on your own infrastructure via Docker. Built-in Tailscale integration provides secure remote access with automatic HTTPS — no port forwarding or cloud services required.

Deploy in seconds

Astro ships as a single Docker image on Docker Hub. One command pulls the latest version, sets up persistent volume mounts for your data, and starts the container. Your notes, documents, and vector store survive upgrades automatically.

Need to update? Run the same command again. It removes the old container, pulls the newest image, and relaunches — your data stays intact.

Installation Guide 
# Install (or update) with one command
curl -fsSL https://raw.githubusercontent.com/marksnyder/astro/main/install.sh | bash

# With Tailscale (first run)
curl -fsSL https://raw.githubusercontent.com/marksnyder/astro/main/install.sh | bash -s -- \
  --ts-authkey tskey-auth-...

# Custom port
curl -fsSL https://raw.githubusercontent.com/marksnyder/astro/main/install.sh | bash -s -- \
  --port 9000
